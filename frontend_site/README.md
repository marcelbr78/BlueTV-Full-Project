# IPTV Buddy - Frontend

Este é o frontend do sistema de gerenciamento IPTV, construído com React, Vite e Tailwind CSS.

## 🛠️ Tecnologias

- **Vite**: Build tool rápida.
- **React**: Biblioteca UI.
- **Tailwind CSS**: Framework de estilização.
- **Shadcn UI**: Componentes de interface.

## 🚀 Como Rodar

### Pré-requisitos
- Node.js (v18+)
- NPM

### Passos

1. **Instalar dependências**:
   ```bash
   npm install
   ```

2. **Rodar servidor de desenvolvimento**:
   ```bash
   npm run dev
   ```

3. **Build para produção**:
   ```bash
   npm run build
   ```

## 📁 Estrutura

- `src/`: Código fonte.
- `src/components/`: Componentes reutilizáveis (UI).
- `src/pages/`: Páginas da aplicação.
